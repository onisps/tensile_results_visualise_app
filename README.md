# Tensile Results Visualizer

Специализированное приложение для анализа, визуализации и обработки данных испытаний на растяжение из файлов Excel.

**[Скачать](https://github.com/onisps/tensile_results_visualise_app/releases/)**

## 🛠 Основной функционал

### 1. Импорт и обработка данных
- **Многопоточная загрузка**: Параллельное чтение файлов `.xls` и `.xlsx` с индикацией прогресса.
- **Интеллектуальный парсинг**: Автоматическое определение заголовков и единиц измерения (`мм`, `%`, `МПа`, `Н`).
- **Конвертация**: Пересчет деформации (`мм` ↔ `%`) и расчет напряжения (`Н` → `МПа`) с использованием начальной площади сечения ($S_0$) из листа «Результаты».

### 2. Визуализация
- **Интерактивные графики**: Подсказки при наведении, настройка стилей линий, толщины и маркеров.
- **Сворачиваемый интерфейс**: Компактное управление настройками («Сортировка», «Оси», «Усреднение»).
- **Масштабирование**: Линейный/логарифмический масштаб, сетка и кнопка быстрого сброса масштаба к началу координат.

### 3. Усреднение
- **Интерполяционное среднее**: Построение точных средних линий для групп образцов (по файлам).
- **Настройка стилей**: Индивидуальное управление цветом и типом линии для усредненных кривых.
- **Режим акцента**: Автоматическая прозрачность (`alpha`) для основных графиков при включении усреднения.

### 4. Сохранение результатов
- **Экспорт графиков**: Сохранение в `PNG` и `PDF` с настраиваемым `DPI` и размерами.
- **Сессии (.tvis)**: Полное сохранение состояния работы в один файл для последующего восстановления.

## 📦 Установка и запуск (для разработчиков)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/onisps/tensile_results_visualise_app.git
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python app.py
   ```

## 🔨 Сборка EXE
Для сборки исполняемого файла используйте:
```bash
python build_exe.py
```
Результат будет в папке `dist/`.
